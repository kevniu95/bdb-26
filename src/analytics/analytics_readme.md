# Kaggle Big Data Bowl 2026 - Analytics

## A. Overview
### `/analytics` repo
This repo contains the notebooks needed to conduct analysis in the corresponding Big Data Bowl [submission](https://www.kaggle.com/competitions/nfl-big-data-bowl-2026-analytics/writeups/new-writeup-1765684019248)

To re-produce analysis from end-to-end, run the notebooks in the following order

1. nfl-big-data-bowl-geometric-gnn-lb-58-in_kaggle.ipynb
   - Run this in Kaggle to leverage GPU usage
2. nfl-big-data-bowl-geometric-gnn-lb-58-in_local.ipynb
3. pred_catch_clean.ipynb
4. pred_catch_analysis.ipynb
5. plays_animated.ipynb
